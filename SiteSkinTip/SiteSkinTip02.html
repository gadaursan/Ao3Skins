Ever since 2021, Ao3 has had a limit of 75 tags per work, but that's still a lot of tags that can take up a lot of your screen space. So it can be very headache inducing trying to scroll through a page, and you get fics with a lot of tags so it looks like the image below (this is an example draft I made up).

It doesn't look so bad on desktop, but we all know it would be a nightmare to scroll through on your phone. So, in this chapter, we'll learn how to compress this tag list into something that makes scrolling a lot easier, like so.

Here is also a preview of what it looks like on a mobile screen.

The Code

If you just want the skin to copy and paste for use, here it is:

Light Mode (Ao3 Default)

dd.kudos,
dd.hits {
visibility: hidden;
}

li.blurb .tags {
max-height: 10em;
overflow-y: auto;
overflow-x: hidden;
background-color: #F0F0F0;
}

li.blurb .tags::-webkit-scrollbar {
width: 5px;
}

li.blurb .tags::-webkit-scrollbar-track {
background: #F0F0F0;
}

li.blurb .tags::-webkit-scrollbar-thumb {
background: #A8A8A8;
}

li.blurb .tags::-webkit-scrollbar-thumb:hover {
background: #707070;
}

Dark Mode (Ao3 Reversi)

li.blurb .tags {
max-height: 10em;
overflow-y: auto;
overflow-x: hidden;
background-color: #282828;
}

li.blurb .tags::-webkit-scrollbar {
width: 5px;
}

li.blurb .tags::-webkit-scrollbar-track {
background: #282828;
}

li.blurb .tags::-webkit-scrollbar-thumb {
background: #606060;
}

li.blurb .tags::-webkit-scrollbar-thumb:hover {
background: #707070;
}

Simply copy paste the Light or Dark mode version into your site skin and you're good to go. If you want to change the colors or the height of the tag box, you can adjust them yourself.
